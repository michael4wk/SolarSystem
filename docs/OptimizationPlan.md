# 太阳系可视化优化改进计划

## 目标与范围
- 提升科普信息的准确与易读性，统一指标口径与单位展示。
- 增强轨道与天体呈现的真实感，同时保持性能与交互体验。
- 建立分阶段的工程实施路线与验收标准，便于团队协作与迭代。

## 信息卡片（四卡布局）
- 温度（橙色）：保留，直观传达环境极端性。
- 自转周期（紫色）：短于约 3 天用“小时”，长于约 3 天用“天”，避免大数字不直观。
- 公转周期（青色）：统一以“地球日”为口径；短于约 2 年用“天”，长于约 2 年用“年”。
- 距离太阳（黄色）：主单位“AU（天文单位）”；辅以“光行程时间”（<60 显示分钟，≥60 显示小时，保留 1 位小数）。

## 单位与口径规则
- 公转“天”即“地球日”的概念（通用科学口径）。
- 1 AU ≈ 8.317 分钟的光行时间；不展示“光年”，避免尺度混淆。
- 中英文一致：小时/天/年、分钟/小时等随语言切换。

## 轨道真实化（圆 → 椭圆）
- 椭圆轨道：太阳位于椭圆的一个焦点，更符合真实且更美观。
- 分阶段实现：
  - 基础版：椭圆形态与焦点偏移，动画常速（不做等面积定律）。
  - 进阶版：近/远日点非匀速（开普勒第二定律近似），提升科学真实度。
  - 高级版：加入轨道倾角与主轴方位角，增强个体差异与识别度。
- 参考偏心率（近似）：水星 0.2056、金星 0.0068、地球 0.0167、火星 0.0934、木星 0.0489、土星 0.0565、天王星 0.0457、海王星 0.0113、冥王星 0.2488。

## 卫星与环的呈现
- 卫星：为主要行星展示代表性卫星（3–5 颗），沿小型椭圆/圆轨道旋转；支持点击查看简介。
- 光环：强化土星光环；木星/天王星/海王星光环以低透明度与细线表现，避免误导。
- 小行星带与柯伊伯带：作为可选开关显示；半透明稀疏点带，默认关闭避免拥挤。

## 交互与 UI 细节
- 透明点击热区：所有小体最小半径约 24–30px，保证易选中。
- 标签抗旋转：标签反向旋转或固定锚点，提升可读性。
- 响应式布局：窄屏两列、中大屏四列，保证美观与信息密度。
- 颜色与图标：温度橙/自转紫/公转青/距离黄，统一编码与风格。

## 数据与结构
- 分层管理：可视化参数（如半长轴、偏心率、动画时长）与科普指标（自转、公转、距离等）分离。
- 国际化：单位与文案随语言切换，确保一致。
- 可拓展：后续可加入平均半径、质量（以地球=1）、卫星总数、典型风暴等。

## 实施阶段
- 阶段 1（快速提升）：
  - 四卡信息展示落地（单位与阈值切换）。
  - 椭圆轨道基础版（焦点偏移、常速）。
  - 土星光环强化、精选卫星可视化。
  - 交互热区与标签可读性优化。

## 验收标准
- 信息卡片单位与阈值切换正确，语言一致。
- 椭圆轨道形态与焦点位置正确，整体观感显著提升。
- 小体点击易用、标签清晰易读。
- 动画流畅（目标 60fps）、网络与密钥安全无泄露。

## 风险与对策
- 视觉拥挤：通过开关控制带状层级与卫星数量；默认开启“精选”元素。
- 性能压力：优先使用合成层动画（transform），在进阶阶段再引入逐帧计算。
- 科普误导：不在行星附近呈现“星云”，以星空/银河背景代替氛围。

## 里程碑与工期建议
 - M1（~1 周）：四卡信息展示落地、椭圆轨道基础版、土星光环强化、精选卫星首批。

## 阶段 2（二维写实升级）
- 目标：在不引入三维与复杂交互的前提下，提升主视图真实感，侧栏展示高清照片作为参考。
- 任务清单：
  1. 接入行星贴图（2K/4K WebP/AVIF，矢量裁剪）
  2. 光照层与边缘优化（terminator / 高光 / 边缘晕影）
  3. 光环材质优化（土星分带，木/天/海低透明度）
  4. 背景星空美化（高分辨底图 + 稀疏星点）
  5. 侧栏高清照片（懒加载、来源说明、独立缩放/平移）
  6. 性能与工程（分档加载、Retina 自适应、资源许可记录）
- 验收：主视图无明显锯齿，移动端流畅（目标 60fps），侧栏高清照片清晰且来源正确。

## 后续拓展清单
- 增加“光谱/化学成分”简表与“表面/大气层”特征图标。
- 引入“对比模式”（如与地球的尺度比较）。
- 提供“学习路径”或“测验模式”，提升互动性。
